## Install and Run

1. Download the repo

```
$ cd neatly-notes
```

2. Install the dependencies:

```
$ npm install
```

3. Run the development server:

```
$ npm start
```

4. Run the json-server:

```
$ json-server --watch data/db.json --port 8000
```

5. Wait for the React to open build environment in development mode in browser
